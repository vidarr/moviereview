This is the
B Movie Review System

Thank you for choosing the perl-driven variant! :)

Everything in here is free. See impressum for further information.

All changeable variables are defined in MovieXHTML.pm (at least nearly ;) ). On the left, you find:
StartThe Start Page ("home") This is basically the file data/start.xhtml. Modify this file acc. to XHTML standard in order to customize your start page!
NewsHere, you find the last NO entries of the rss feed. The basic values are NO = 10, the standard rss file is "mrs.rss". You can alter these values by modifying either within file "news.pl" NO_ENTRIES or within file "MovieXHTML.pm" the value of RSS_FILE.
All MoviesThis gives you a list of all movies registered so far. Choose one to see all details of the movie.
ImpressumThis shows the file data/impressum.xhtml. Alter this file to show whatever kind of impressum you want to use.
RSS feedThis is the rss feed of the page. It is handled atuomatically.

For feeding the system with proper data, the usage of the BMovieReviewer is highly recommended. Tis is a easy to use Java application. Just make it post the data to the location your server can be found. For how to perform this, have a look at the Reviewer documentation or the bare code.

To register new users, use the script "adduser.pl". For example,
./adduser foo password
in order to create a user "foo" with password password. To delete a user, open the file "users.sst" with your favourite editor (I recommend emacs OR vi ;), scroll for the user name in the first column and delete the entire line (dont panic, noone will be able to reconstruct your password from the data found in this file, at least its almost sure).
Initially, each user receives an ordinary description page saying that there is none - not quite satisfying, is it? Yout can modify the user profile by writing nice text into the users/foo.xhtml (replace foo by the actual user name - the file is created by the adduser - script).

All posted movies can be found within the "movies" subdirectory. To delete a movie, remove the xml and jpg file from that directory and check "movies.sst" for any appearance of the movie title. To remove the rss - entry of the movie as well, scan the rss file for any entry containing the movie's title.

I know, by now it's quite a hard job to do any unusual thing like removing anything thing. Probably there will be some enhancements for your comfort.
